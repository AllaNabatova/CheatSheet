CSS Selectors and Specificity

Selectors

Category Description Example
Type Type selectors style HTML elements of a given type or tag name.
h1 { color: white; }
Class Class selectors define elements based on the contents of their class attribute.
.calendar-cell { color: #fff;
}
ID ID selectors match elements based on the contents of their id attribute.
#job-title { font-weight: bold; }

Rules

Rule Description Example
Grouping Multiple selectors can be matched so that same style is applied to them.
h1, h2 { color: red; }
Chaining Chaining allows to apply style only to elements that match all criteria.
/* Select h3 elements with the section-heading class */ h3.section-heading { color: blue; } /* Select elements with the section-heading and button class */ .section-heading.button { cursor: pointer; }
Descending Use descendant selector combinator to style elements that are descended from another selector.
div p { } section ol li { }
Specificity Selectors are applied in such priority:
1. ID selector;
2. Class selector;
